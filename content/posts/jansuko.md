麻雀の点数管理、どうしていますか？

紙に書いたり、スプレッドシートで計算したり…。ゲームが終わるたびに電卓を叩いて精算額を計算するのは、正直面倒ですよね。

そこで、麻雀の点数記録・精算を簡単に管理できるWebアプリ「**じゃんスコ**」を作りました。

## 目次

## じゃんスコとは

じゃんスコは、麻雀の点数記録に特化したWebアプリです。

ゲームごとに各プレイヤーの点数を入力するだけで、**合計点・増減・精算額・順位**をリアルタイムに自動計算してくれます。

![セクション詳細](/posts/セクション詳細.png)

スプレッドシートで毎回数式を組む必要も、電卓で精算額を計算する必要もありません。

## コンセプト

「入力は点数だけ。あとは全部自動」を一番大事にして作りました。

麻雀が終わったあとの精算は、なるべくスムーズに済ませたいもの。複雑な操作は要らない。点数を入力したら、そのまま結果が出てくる。そんなシンプルさを目指しています。

麻雀卓の周りでスマホを開いて使うことを想定して、モバイルファーストで設計しているのもこだわりのひとつです。

## 主な機能

### セクション管理

「セクション」とは、1回の麻雀セッション（複数ゲームのまとまり）を管理する単位です。

例えば「2024年1月1日の新年会」のように、その日の対局をまとめて記録できます。

![セクション一覧](/posts/セクション一覧.png)

セクション作成時に設定するのは3つだけ：

- **参加者**（3〜4人）
- **開始点**（例: 35,000点）
- **レート**（例: 50円/1,000点）

### 点数入力と自動計算

半荘が終わるたびに各プレイヤーの点数を入力するだけ。

![点数入力画面](/posts/点数入力.png)

入力した点数から以下が自動で計算されます：

| 項目   | 説明                           |
| ------ | ------------------------------ |
| 合計点 | 各ゲームの点数の総和           |
| 増減   | 合計点 − (開始点 × ゲーム数)   |
| 精算額 | 増減 × レート ÷ 1,000          |
| 順位   | セクション内の合計点による順位 |

点数の合計が「参加人数 × 開始点」と一致しない場合はバリデーションエラーが表示されるので、入力ミスも防げます。

### 統計機能

プレイヤーごとの成績を振り返ることもできます。

![統計](/posts/統計.png)

確認できる指標：

- 勝率
- 平均順位
- 通算収支
- 参加ゲーム数

期間指定（全期間/月別/カスタム）にも対応しているので、「先月の成績はどうだったかな？」といった振り返りも簡単です。

## 計算例

実際の計算例を見てみましょう。

**条件**: 開始点35,000点、3人麻雀、レート50円/1,000点、5ゲーム実施

|           |  かいと |  きむら |   えいご |
| --------- | ------: | ------: | -------: |
| 1ゲーム目 | -15,000 |  61,000 |   59,000 |
| 2ゲーム目 |  57,000 |  50,000 |   -2,000 |
| 3ゲーム目 |  75,000 |  36,000 |   -6,000 |
| 4ゲーム目 |  62,000 |  56,000 |  -13,000 |
| 5ゲーム目 |  70,000 |  27,000 |    8,000 |
| **合計**  | 249,000 | 230,000 |   46,000 |
| **増減**  | +74,000 | +55,000 | -129,000 |
| **精算**  | +¥3,700 | +¥2,750 |  -¥6,450 |
| **順位**  |     1位 |     2位 |      3位 |

これらの計算がすべて自動で行われます。

## こんな人におすすめ

- 仲間内で定期的に麻雀をしている人
- 毎回の精算計算が面倒に感じている人
- 自分の成績を記録して振り返りたい人

3人麻雀・4人麻雀どちらにも対応しています。スマホからもPCからも使えます。

## 今後の予定

現在のバージョンでは以下の機能が未対応です。将来的に対応を検討しています：

- **返し点**: 精算時の基準点（オカ計算）
- **ウマ**: 順位によるボーナス・ペナルティ
- **データエクスポート**: CSV等への出力

## おわりに

「じゃんスコ」は、麻雀の点数管理をシンプルにするために作ったアプリです。

点数を入力するだけで精算まで自動で出てくるので、ゲームが終わったあとの手間がなくなります。仲間内での麻雀がもっと楽しくなれば嬉しいです。
